# Java-Basic-Pharmacy-GUI

## About 
This is a simple Pharmacy GUI created using Swing. It implements the CRUD functionalities.

## Features
- Signing up and Logging In<br>
- Adding drug records<br>
- Searching and editting drug records<br>
- Deleting drug records<br>

## Technologies Used
- Java Swing<br>
- MySQL<br>

## Future Improvements
- A centralized dashboard to replace the need for different pages for functionalities
- An admin dashboard to manage employees and all drug info
